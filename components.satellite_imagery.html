  • path (str) – Path of the input .tif image

  • parts (int) – Splits the single image into smaller images with a total of (parts * parts) images returned

>>>        path="sample.tif", parts=5
>>> )
>>> si.split()

components.satellite_imagery.utils.add_padding(image, padding_height, padding_width, color=(0, 0, 0))¶

add padding around the image so that buildings touching the corners can be bought to center create new image of desired size and color (black) for padding This is necessary as corner buildings will not be detected by countour detection algorithm if the building edges are outsite of the image

Parameters

  • images – Image that you want padding around

  • padding_height (int) – Amount of padding in y direction

  • padding_width – Amount of padding in x direction

  • padding_width – int

  • color (1x3 tuple) – RGB color of the padding

Returns

Image with padding
